Privacy Policy

Last updated: March 14, 2026

1. Introduction

This Privacy Policy describes the data handling practices for the Market Trends Explorer application ("Application"). This policy is provided to comply with the Login with Amazon (LWA) security profile requirements. This Application is built strictly for internal personal research and company use. It is not available to the public, and no external users are permitted to log in or use the Application.

2. Information We Access

Since this is an internal tool, the Application only accesses the Amazon account information of the authorized internal developer(s) and company account(s) used for API integration.

  • Authentication: We use Login with Amazon (LWA) solely to securely authenticate our own internal company accounts to access Amazon's APIs.
  • API Data: We retrieve data via Amazon Advertising and Selling Partner APIs strictly for internal market research, trend analysis, and advertising optimization.

3. Data Usage and Security

Because the Application is not public, we do not collect, process, or store any personal data from external users or the general public. All internal data accessed through the Amazon APIs is:

  • Used exclusively for internal company research and operational purposes.
  • Stored securely on our internal servers.
  • Never sold, distributed, or shared with third parties.

4. Contact

This Application operates internally. If you are an auditor or Amazon representative reviewing this policy for compliance, please contact the developer associated with the Login with Amazon security profile for any further inquiries.
